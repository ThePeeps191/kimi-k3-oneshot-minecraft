# Kimi K3 One-shot Minecraft

Kimi K3 was provided the prompt in `prompt.txt`, and the output file is at `index.html`. This was generated with Kimi K3 Max within mobile chat app, where K3 was first deployed live, and is one of the first examples of code generated by the new Kimi K3 model releasing today (July 16 2026).

Play the game live [here](https://thepeeps191.github.io/kimi-k3-oneshot-minecraft/).
